# Генератор паролей

Простая консольная программа на Python для генерации случайных безопасных паролей. Поддерживает пароли с буквами, цифрами и специальными символами.

## Особенности

- Генерация паролей произвольной длины.
- Используются символы:
  - Заглавные буквы: `A-Z`
  - Строчные буквы: `a-z`
  - Цифры: `0-9`
  - Специальные символы: `!@#$%^&*()_-+=<>?/{}~|`
- По умолчанию длина пароля — 12 символов.

## Требования

Для запуска программы требуется Python 3.6 или выше.

## Установка

1. Скачайте файл программы `RPG.py`.
2. Убедитесь, что у вас установлен Python 3.6 или выше.
